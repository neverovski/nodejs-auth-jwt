### Before starting - dev stage
```bash
$ cp .env.example .env
$ docker-compose up -d --build
$ npm run start:dev
```

### Before starting - prod stage
```bash
$ cp .env.example .env
$ docker-compose up -d --build
$ npm run start:prod
```
